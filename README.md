# Bamboo Immortals · 竹林修仙传

> 竹林之中，修心养性。A Chinese-style goal automation system — cultivate discipline through daily reflection.

## 中文

**竹林修仙传** 是一款基于苏联控制论之父维克托·格卢什科夫提出的「OGAS」理念、专为个人打造的中国风目标自动化分配管理系统。它帮你以每日复盘为核心，系统化管理目标、任务与情绪。

### 功能

- **每日复盘**：记录情绪、任务、反思，竹影婆娑的东方美学界面
- **目标管理**：OGAS 式的目标拆解与优先级分配
- **主题特效**：可自定义色相/明度的动效主题
- **白噪音**：内置竹林氛围音（流水、蝉鸣、竹琴、鸟鸣、夏夜）
- **板块配置**：自由拖拽排序、显隐仪表盘各模块
- **调色联动**：可选将 webapp 调色同步到 Obsidian 原生界面

### 使用方法

1. 点击左侧 Ribbon 栏的叶子图标，或命令面板中运行「打开今日复盘」
2. 通过右下角悬浮按钮（FAB）访问设置、主题、工具
3. 写每日复盘、追踪目标，沉浸竹林意境

### 设置

- **数据路径**：复盘数据在 Vault 中的存储目录
- **主题动效**：各板块的自定义视觉效果
- **白噪音**：配置音源文件夹
- **调色联动**：色相/明度调整同步到 Obsidian 原生 UI

## English

**Bamboo Immortals** is a Chinese-style goal automation system inspired by the OGAS cybernetics concept proposed by Viktor Glushkov. It centers on daily reviews to systematically manage goals, tasks, and moods.

### Features

- **Daily Review**: Track moods, tasks, and reflections in a bamboo-themed interface
- **Goal Management**: OGAS-inspired goal breakdown and prioritization
- **Custom Themes**: Visual effects with adjustable hue and lightness
- **Ambient Sounds**: Built-in bamboo forest atmospheres (stream, cicada, bamboo zither, birdsong, summer night)
- **Section Configuration**: Drag to reorder and toggle dashboard modules
- **Theme Sync**: Optionally sync palette adjustments to Obsidian's native UI

### Usage

1. Click the leaf icon in the left ribbon, or run "Open daily review" from the command palette
2. Use the floating action button (FAB) to access settings, themes, and tools
3. Write your daily review, track goals, and immerse in bamboo aesthetics

### Settings

- **Data path**: Where review data is stored in your vault
- **Theme effects**: Custom visual effects for each section
- **Ambient sounds**: Configure white noise source folders
- **Palette sync**: Sync color adjustments to Obsidian's native interface
